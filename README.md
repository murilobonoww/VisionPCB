# VisionPCB
Projeto de TCC do curso de Análise e Desenvolvimento de Sistemas. Realizado entre 4 pessoas, o VisionPCB é um software para um sistema embarcado que tem como objetivo aumentar a eficiência nas indústrias enquanto diminui erros humanos. O sistema realiza uma inspeção automatizada de placas (ou qualquer peça que seja cadastrada), para garantir a montagem correta dos componentes.
